# 🎡 青溪國小 114 學年度親職教育日 (Qingxi Elementary School Fair)

這是一個專為青溪國小園遊會設計的資訊服務平台，旨在提供家長與學生最即時、便利的活動資訊。

🔗 **線上預覽 (Live Demo)**: [https://gn00318361-collab.github.io/csps_site/](https://gn00318361-collab.github.io/csps_site/)

## ✨ 特色功能 (Features)

*   **📱 行動裝置優化 (Mobile First)**：專為手機瀏覽設計，攤位列表自動轉為卡片式，方便現場查找。
*   **⏳ 活動倒數計時**：首頁動態顯示距離活動開始的剩餘時間，增加期待感。
*   **📢 即時跑馬燈**：置頂公告重要訊息（如雨備方案、開幕時間）。
*   **🔍 智慧搜尋與篩選**：
    *   關鍵字搜尋（如：502、炸雞、遊戲）。
    *   類別篩選按鈕（😋 食物、🛍️ 物品、🎮 遊戲）。
*   **🗺️ 校園地圖整合**：直接嵌入校園平面圖，並提供高清版連結。
*   **⬆ 一鍵回到頂部**：長列表瀏覽更友善。

## 🛠️ 技術 (Tech Stack)

*   **HTML5 / CSS3**：原生語法，無須編譯，輕量快速。
*   **JavaScript (Vanilla)**：處理搜尋邏輯、倒數計時與介面互動。
*   **GitHub Pages**：靜態網頁託管。
